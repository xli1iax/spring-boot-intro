# Spring Boot Backend Project

## Description
Backend REST API project built with Spring Boot.
The project demonstrates layered architecture and basic CRUD operations.
Includes basic configuration and database integration.

## Technologies
- Java 17
- Spring Boot
- Spring Data JPA
- Maven
- REST API
- DTO pattern
- Layered architecture (Controller, Service, Repository)
- Exception handling

## Project Structure
- controller – REST endpoints
- service – business logic
- repository – database access layer
- model – entity classes
- dto – data transfer objects
- mapper – mapping between entity and DTO
- exception – custom exception handling
- config – configuration classes

## How to Run
1. Clone the repository
2. Run `mvn spring-boot:run`
3. Application starts on default port 8080
